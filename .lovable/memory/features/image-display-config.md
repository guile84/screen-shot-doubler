---
name: Image Display Config
description: Per-image object-fit and focal point configuration in media table
type: feature
---
- Media table columns: object_fit (cover/contain, default cover), focal_x (0-1), focal_y (0-1)
- ImageUploader has settings button per image → dialog with fit selector + click-to-set focal point
- Public product page and portfolio use object_fit + focal point for rendering
